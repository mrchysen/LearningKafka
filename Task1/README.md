# Task 1 - "Hello, Kafka!" — Basic Interaction

Objective: Understand key concepts: Producer, Consumer, Topic, Broker, Message.

Tasks:

Create a topic manually (via Kafka CLI in the container): test-messages.

Write a simple Producer that sends strings entered by the user in the console to the topic.

Write a simple Consumer that reads all messages from the beginning of the topic and prints them to the console.

Run the Producer and Consumer simultaneously in different console applications. Verify that the Consumer receives new messages in real time.

What you will learn: ProducerConfig, ConsumerConfig, basic methods ProduceAsync and Subscribe, GroupId.
